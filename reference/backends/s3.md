# S3 对象存储

适用于：

- 公有云对象存储（如 AWS S3、阿里云 OSS、腾讯云 COS 等），需提供 S3 兼容接口；
- 自建的 S3 兼容对象存储集群。

在表单中，你需要重点关注以下字段：

- **Bucket**（必填）
  - 存放对象的目标桶名称，例如：`my-bucket`。
  - 建议使用专门为 WANOS 准备的桶，避免与其他业务混用权限复杂。

- **Access Key / Secret Key**（必填）
  - 对应云厂商或自建 S3 的访问密钥对。
  - 建议为 WANOS 创建专用账号，并限制为最小必要权限。

- **Endpoint**
  - S3 服务访问地址，例如：`https://s3.amazonaws.com`、`https://oss-cn-beijing.aliyuncs.com`。

- **Base Path** / **Base Path（前缀路径，可选）**
  - 如果希望所有对象都存放在桶内的某个前缀下（例如 `wanos/`），可以在此填写前缀路径。
  - 留空表示直接存放在桶根目录下。

> 填写完成后，建议先点击「测试连接」，确认凭证和 Endpoint 配置正确再保存。
